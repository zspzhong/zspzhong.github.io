  • Hexo+Github 搭建博客

    2017-07-27 |
    • { Hexo }
    • { Github }

    很多大神都在Github搭建博客，我就写一个自己搭建的流程，其实搭建起来很快，一下子就搭好了。博客这个东西还是要靠更新。

    前期准备

    1.注册gihub账户
    2.安装node环境
    3.安装git

    正式开始

    1.本地安装Hexo

    $ npm install hexo-cli -g

    2.初始化blog项目文件

    $ hexo init blog

    3.安装blog依赖

    $ cd blog
    $ npm install
    $ hexo server或$ hexo s
    这个时候访问 http://localhost:4000 就可以看到搭建的博客了
    就是样子有点丑
